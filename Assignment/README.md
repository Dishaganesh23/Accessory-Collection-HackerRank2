# DAA Assignment – Accessory Collection

## Problem
Accessory Collection (HackerRank)

Victoria wants to buy L accessories from a store that has A different types of accessories.
Each accessory type has a cost equal to its type number.

She must buy the accessories such that any subset of N accessories contains at least D distinct types.

The goal is to find the maximum amount of money she can spend while satisfying the rule.

C++
